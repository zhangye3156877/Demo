JS条件判断：if转换

    非boolean类型会转换成boolean类型判断

  • undefined => false
  • null => false
  • string 空字符串 => false , 其他 => true
  • number 0 -0 +0 NaN => false, 其他 => true
  • Object => true

JS == 判断

    typeof A 与 typeof B 相同

  • 类型为 undefined 或 null 时 =>true
  • 类型为 number 任意一方为 NaN 或 数值不等时 => false , 数值相等时 => true(+0,0,-0完全相等)
  • 类型为 string 完全一致时 => true
  • 类型为 object 引用一致 => true

    • typeof A 与 typeof B 不同

  • A 为 null B 为 undefined => true
  • A 为 number B 为 string 比较 A == Number(B)
  • A 为 boolean B 为非 boolean 比较 Number(A) == B
  • A 为 object B 为 string 或 number 比较 toPrimitive(A) == B
    (toPrimitive为内部方法.
    1.如果输入值是原始值，直接返回这个值；
    2. 否则，如果是对象，调用valueOf()，如果结果是原始值，返回结果；
    3. 否则，调用toString()。如果结果是原始值，返回结果；
    4. 否则，抛出错误。)
  • 其他情况 => false
